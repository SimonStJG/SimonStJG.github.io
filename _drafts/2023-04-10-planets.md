## Ingredients

* Suspiciously cheap Arduino Nano ?Knockoff from eBay - only £5 each!  "Arduino Nano with USB-C or Micro USB port Atmega328P"
* "Flat Top WIde Angle LEDs 3mm/5mm : Red,Blue,Green,Yellow,White,Orange,Pi.." also from eBay's phi-sigma-electronics.  Got the 5mm Yellow White.  The forward voltage was measured at 2.95V and it looks good with ~20mA.
* "TracoPower TSR 1-2450 Single Output DC-DC Converter 1W" from eBay.
* NPN TO-92 transitors which I had lying around anyway.

## Circuit

This circuit works really well, using the Arduino PWM to drive the LEDs through an NPN transitor.  I actually used all four PWM outs (6, 9, 10, 11)so I could get the LEDs to fade in groups.  Nothing clever with the PWM out.

   OCR0A: TC0 Output Compare Register A

  OCR0A = 0xFF;

   TIMSK0: TC0 Interrupt Mask Register
   https:onlinedocs.microchip.com/pr/GUID-0EC909F9-8FB7-46B2-BF4B-05290662B5C3-en-US-12.1.1/index.html?GUID-6F9AB112-70B4-4C29-8DA8-D42C55980540
   Bit 1 – OCIE0A: Timer/Counter0, Output Compare A Match Interrupt Enable
   
   In other words, when Timer0 is above the value in OCR0A, trigger the handler TIMER0_COMPA_vect
  TIMSK0 |= _BV(OCIE0A);
}

void loop() {
   Nothing to do here, it's all interrupts
}

/**
 * Timer0 interrupt handler.
 * "TimerCounter0 Compare Match A"
 * 
 * < 1.5s
 *   fade out then fade back in
 * < 8s
 *   constant light
 */
SIGNAL(TIMER0_COMPA_vect) 
{
  millisSinceLastUpdate += 1;
  if (millisSinceLastUpdate < 1500) {
    analogWrite(PIN_LED_0, pgm_read_byte_near(&fadeOutFadeIn[millisSinceLastUpdate]));
   
  } else {
    analogWrite(PIN_LED_0, 255);
  }

  if (millisSinceLastUpdate >= 1500 && millisSinceLastUpdate < 3000) {
    analogWrite(PIN_LED_1, pgm_read_byte_near(&fadeOutFadeIn[millisSinceLastUpdate - 1500]));
  } else {
    analogWrite(PIN_LED_1, 255);    
  }

  if (millisSinceLastUpdate >= 3000 && millisSinceLastUpdate < 4500) {
    analogWrite(PIN_LED_2, pgm_read_byte_near(&fadeOutFadeIn[millisSinceLastUpdate - 3000]));
  } else {
    analogWrite(PIN_LED_2, 255);    
  }

  
  if (millisSinceLastUpdate >= 4500 && millisSinceLastUpdate < 6000) {
    analogWrite(PIN_LED_3, pgm_read_byte_near(&fadeOutFadeIn[millisSinceLastUpdate - 4500]));
  } else {
    analogWrite(PIN_LED_3, 255);    
  }
 
  if (millisSinceLastUpdate > 9000) {
    millisSinceLastUpdate = 0;
  }
}
